# IAM基礎・実務ガイド（BoK）

> 日本向け・初学者向け・最新論点対応の IAM BoK を、ゼロから設計・執筆するプロジェクトページです。

## この BoK の目的

- 初学者が前から順に学べる
- 日本の制度・実務に接続している
- Passkeys / NHI / AIエージェント / VC など新領域を本論で扱う
- 導入・運用・監査・インシデント対応まで一体で理解できる

## 構成（全6部・34章）

1. **第1部 入門**（用語・概念・業務シナリオ）
2. **第2部 認証・認可の基礎技術**（MFA, SSO, OAuth, OIDC, PKCE, Passkeys）
3. **第3部 設計と運用**（アーキテクチャ、ライフサイクル、IGA、監査、事故対応）
4. **第4部 日本実務**（個人情報保護、公的本人確認、eKYC、業界差分）
5. **第5部 新領域**（NHI、ワークロードID、AIエージェント、VC、倫理）
6. **第6部 実践**（導入ロードマップ、RFP、体制、KPI、ケーススタディ）
